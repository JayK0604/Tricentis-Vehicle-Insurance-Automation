# Tricentis-Vehicle-Insurance-Automation

## Overview

This project automates the Tricentis Vehicle Insurance web application using Selenium WebDriver and Java. The automation covers multiple insurance modules and validates form submission workflows.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Eclipse IDE
- Git & GitHub

## Modules Automated
- Automobile
- Truck
- Motorcycle
- Camper

## Features
- Automated form filling
- Validation of user inputs
- Screenshot capture on successful execution
- End-to-end workflow automation

## Prerequisites
- Java JDK 8+
- Eclipse IDE
- Selenium WebDriver
- Chrome Browser

## How to Run
1. Clone the repository
2. Open the project in Eclipse
3. Configure ChromeDriver
4. Run the test class
